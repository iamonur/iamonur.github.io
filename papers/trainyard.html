  • Level Generator:
    • Mission Generator: Determines the stations and trains to be used by color and count.
    • Grid Placement: Makes sure that every station is reachable by the corresponding train. However, this is not enough to say the level is solvable.
    • Here is how the level generator works:
  • Level Solver: Truly checks if a level is solvable by trying one-by-one. If it is solvable, records the easiest solution of it. Here's how level solver works:
  • Difficulty Estimation: Difficulty is tied up to 5 different factors listed below(All weights are calculated from a user study):
    1. Stations(weighted 0.1601): Number of stations increase difficulty.
    2. Trains(weighted 0.1126): Difference between start and end number of trains increase difficulty.
    3. Colors(weighted 0.2194): Symmetric difference between the start and the end colors mean switching should be made for every time and this increases difficulty.
    4. Tracks(weighted 0.4267): Number of switches in the simplest solution increases difficulty.
    5. Grid(weighted 0.1177): Empty grid space to be worked on to solve the level increases difficulty.
