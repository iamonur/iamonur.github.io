A brief summary

  • Abstract
    • Since Multiple Classifier Systems(MCS) is an important path in ML, it becomes important to choose a combination.
    • Classificational Cellular Automata(CCA) is proposed in this paper.
  • Introduction
    • It is hard to choose a model for ML, and the success depends on it. This is mostly done via experience.
    • CCA both takes advantage of the MCS, and the self-organizing abilities of CA.
    • In CCA, multiple ML approaches are implemented.
  • Multiple Classifier Systems
    • Also known as ensembles, or committees.
    • Since each classifier has some weak sides, combining them should improve the performance.
    • One of the most popular: Bootstrap Aggregation - Bagging and Boosting.
    • Bagging: Replicates training set and trains agents alone. Final classification is done by a majority vote.
    • Boosting: Combines classifiers with weighted voting. Distribution of training set is done adaptively.
  • Cellular Automata
    • CA are both simple and models complex systems.
    • Spatially extended decentralized system made up of individual cells.
    • Each cell is in a specific state that changes each step.
    • CA as a MCS Model
      • MCS models are either work on combination, or perturbation.
  • Classificational Cellular Automata
    • CCA is presented as a classifier.
    • Each cell can contain a classifier, and a state according to its classification of an input sample. This state can be same as the sample's class, a different class, or a "cannot be classified.".
